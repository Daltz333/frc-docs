Getting Started with Romi
=========================

.. warning:: The Romi documentation for WPILib is WIP!

`Romi <https://www.pololu.com/category/202/romi-chassis-and-accessories>`__ is a differential-drive robot platform with the following capabilities.

- 2WD drive base with fixed ball caster
- `120:1 DC gear motors <https://www.pololu.com/product/1520>`__
- `12 CPR quadrature encoders <https://www.pololu.com/product/3542>`__ attached to each drive motor
- Arduino compatible ATMega32U4 controller
- On-board 3-axis accelerometer and gyro
- On-board push buttons and LEDs
- Variety of GPIO pins
- Connector for Raspberry Pi
